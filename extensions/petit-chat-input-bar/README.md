# petit-chat-input-bar

A tiny static companion sprite above the editor (a la Vibe's petit chat).

Renders a small ASCII pet that sits above the input bar in its neutral pose.
Deliberately static — periodic TUI renders reset the scrollback viewport to
the bottom, so animation would yank the view down.

```
  ⡠⣒⠄  ⡔⢄⠔⡄
 ⢸⠸⣀⡔⢉⠱⣃⡢⣂⡣
  ⠉⠒⠣⠤⠵⠤⠬⠮⠆
```

Purely cosmetic. No commands, no config.
